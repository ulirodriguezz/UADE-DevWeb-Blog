Sobre mi

Mi nombre es Ulises Rodriguez, tengo 24 años y soy estudiante de Ingeniería Informática en UADE. Actualmente estoy cursando el quinto año de la carrera, donde he podido profundizar en áreas como arquitectura de software, bases de datos, desarrollo web y aplicaciones distribuidas.
Me desempeño como desarrollador Fullstack, con experiencia en la creación de aplicaciones web utilizando Spring Boot para el backend y React para el frontend, trabajando principalmente con los lenguajes Java y JavaScript.
Este blog forma parte de un proyecto académico para la materia Desarrollo Web.
En esta página se utiliza flexbox y grids en conjunto con media queries y funciones clamp, además de unidades de medida relativas, para crear una pagina responsiva que se adapte a distintos tamaños de pantalla. Ya sea PCs, Tablets o Moviles.
A continuación, encontraras una sección de articulos con algunas noticias interesantes y una sección de contacto con un formulario para que me dejes un mensaje. Podes navegar a las distintas secciones por medio del scroll bar o haciendo click en los titulos del navbar
